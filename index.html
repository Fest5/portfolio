Projects i worked on:

Explorify - Book the Best Tours on the USA!

Explorify is a web application where you can buy tours on multiple locations on the USA. The page has a nice Front End design, but most importantly it has a lot of functionalities made possible thanks to its own made API. This page was made with JavaScript, Node.js, Express, MongoDB for the database, mongoose, pug, JWT, webpack and a lot more modern technologies (you can see them all on package.json file). Click here to go to the page! and here to see the code

Forkifyx - Search over a million recipes!

Forkifyx is a web application where you can search over a million recipes that you wish. It is a SPA and it was made with Advanced Javascript, consuming the Spoonacular API. The page is responsive and it has a lot of functionalities like selecting a recipe, see its details, add a recipe to your favorites that are saved with LocalHost, add ingredients to your shopping cart and more. Click here to go to try this page and here to see the code

Discover - Find Amazing Tours in Argentina!

Discover is a responsive web page where you can choose a new location in Argentina to have a journey. It's focused on the design, with multiple animations, a cool menu and interactive buttons. It was made with advanced Sass, HTML and CSS. Click here to go to the page! and here to see the code
